# Wechat Moments

Wechat Moments `React` implement, Based on `Create React App`.

## OS Requirements
- unix-like os
- nodejs >= 10
- yarn

## Dependencies
- react
- redux/react-redux
- photoswipe

## How to run
```
git clone https://github.com/oluul/wechat-moments --depth 1
cd wechat-moments

yarn install
yarn start
```

## Build
```
npm run build
```
